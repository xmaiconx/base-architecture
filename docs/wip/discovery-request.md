## Descrição sobre a funcionalidade

Desenvolva a funcionalidade de conversão de mídia para texto (audio e imagem) utilizando OpenAI.
Inclua uma variavel OPENAI_API_KEY no arquivo @apps\backend\.env.example  para que possamos utilizar a API do OpenAI.
